# Responsive Admin Dashboard Ecommerce-site
A modern Responsive Admin Dashboard Ecommerce-site build using HTML , CSS AND JAVASCRIPT. 
It includes a clean UI, analytics cards, recent orders table and light/dark mode.

# 🚀 Features
- Responsive layout
- Admin dashboard UI
- Sales analytics cards
- Recent orders table
- Light / Dark theme toggle
- Clean and modern design
  
# 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

# 📂 Project Structure 
images -> dashboard images
index.html -> main page
style.css -> styling file
index.js -> dashboard logic
orders.js -> orders data
